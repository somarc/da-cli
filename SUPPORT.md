# Support

Use GitHub Issues for public support.

- Bug reports: include OS, Node.js version, `da --version`, the command you ran, expected output, and actual output.
- Questions: include the DA org/repo shape only when it is safe to share publicly.
- Security issues: follow [SECURITY.md](./SECURITY.md) instead of opening a public issue.

Remove tokens, bearer headers, private URLs, and customer data from logs before posting.
