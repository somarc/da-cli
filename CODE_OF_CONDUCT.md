# Code of Conduct

This project follows the [Contributor Covenant Code of Conduct](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

Be respectful, constructive, and focused on improving the project. Harassment, abuse, and intentionally disruptive behavior are not acceptable in issues, pull requests, discussions, or other project spaces.

Instances of unacceptable behavior may be reported to the repository owner through GitHub. Maintainers may remove comments, close threads, block users, or take other reasonable action to protect the project community.
